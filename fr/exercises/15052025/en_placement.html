Règles de Placement du Pronom “en”

Le pronom “en” remplace un nom introduit par “de” (de, du, de la, des) ou une quantité. Il répond à des questions comme “de cela ?”, “en as-tu ?”, etc.

Principales règles :
1. Temps simples : “en” avant le verbe (J’en parle).
2. Temps composés : “en” avant l’auxiliaire, pas d’accord (J’en ai parlé).
3. Deux verbes : “en” avant l’infinitif (Je veux en parler).
4. Impératif :
- Affirmatif : après le verbe avec trait d’union (Parles-en).
- Négatif : avant le verbe (N’en parle pas).

Exercices

1. Temps Simples (Présent/Futur/Imparfait)

Remplacez la partie entre crochets par “en” :
1. Je parle [de mon projet] → J’ parle.
2. Nous achetons [des pommes] → Nous achetons.
3. Tu prendras [du café] → Tu prendras.
4. Elle mangeait [des bonbons] → Elle mangeait.
5. Vous boirez [de l’eau] → Vous boirez.
6. Ils parlent [de leurs vacances] → Ils parlent.
7. Nous avions [des idées] → Nous avions.
8. Elle achètera [des livres] → Elle achètera.
9. Tu veux [du pain] → Tu veux.
10. Ils recevaient [des cadeaux] → Ils recevaient.

2. Temps Composés (Passé Composé)

Remplacez par “en” :
1. J’ai parlé [de ce problème] → J’ parlé.
2. Ils ont acheté [des fruits] → Ils acheté.
3. Tu as bu [du lait] → Tu bu.
4. Nous avons reçu [des lettres] → Nous reçu.
5. Elle a pris [des photos] → Elle pris.
6. Vous avez parlé [de vos projets] → Vous parlé.
7. Il a mangé [du chocolat] → Il mangé.
8. Elles ont acheté [des robes] → Elles acheté.
9. J’ai reçu [des messages] → J’ reçu.
10. Nous avons bu [du jus] → Nous bu.

3. Deux Verbes (Verbe + Infinitif)

Placez “en” correctement :
1. Je veux acheter [des pommes] → Je veux acheter.
2. Elle peut prendre [du sucre] → Elle peut prendre.
3. Nous allons manger [des crêpes] → Nous allons manger.
4. Tu dois boire [de l’eau] → Tu dois boire.
5. Ils veulent parler [de leur projet] → Ils veulent parler.
6. Vous pouvez acheter [des billets] → Vous pouvez acheter.
7. Je vais prendre [du pain] → Je vais prendre.
8. Elle souhaite recevoir [des cadeaux] → Elle souhaite recevoir.
9. Nous préférons acheter [des légumes] → Nous préférons acheter.
10. Ils aiment boire [du thé] → Ils aiment boire.

4. Impératif (Affirmatif/Négatif)

Transformez en commandes avec “en” :
1. Acheter [des pommes] ! (affirmatif) → !
2. Ne pas prendre [de sucre] ! (négatif) → !
3. Manger [des fruits] ! (affirmatif) → !
4. Ne pas boire [de café] ! (négatif) → !
5. Prendre [du pain] ! (affirmatif) → !
6. Ne pas acheter [de billets] ! (négatif) → !
7. Parler [de ce sujet] ! (affirmatif) → !
8. Ne pas recevoir [de cadeaux] ! (négatif) → !
9. Acheter [des légumes] ! (affirmatif) → !
10. Ne pas parler [de ce problème] ! (négatif) → !

Pour consolider votre apprentissage

Rédigez un court paragraphe (3-5 phrases) expliquant ce que vous avez retenu sur le placement du pronom “en” en français. Utilisez vos propres mots et, si possible, donnez un exemple.

Notes :

  • Pas d’accord : Le participe passé ne s’accorde pas avec “en” (“J’en ai mangé”).
  • Trait d’union à l’impératif affirmatif seulement (Prends-en vs. N’en prends pas).
